# An S4 class to represent a DOT object.

An S4 class to represent a DOT object.

## Slots

- `srt`:

  A list containing spatial data as processed by setup.srt

- `ref`:

  A list containing reference data as processed by setup.ref

- `weights`:

  A matrix containing absolute abundance of categories

- `solution`:

  A matrix containing raw solution

- `history`:

  A data.frame containing solution history
